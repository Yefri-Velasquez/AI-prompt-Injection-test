Getting Started with Spring Boot

March 30, 2026 · 7 min read

Prerequisites

  • Java 17 or later installed
  • An IDE (IntelliJ IDEA, VS Code, or Eclipse)
  • Maven or Gradle (Maven is used in this guide)

Step 1: Generate a Project with Spring Initializr

The easiest way to start is with start.spring.io. Configure your project:

  • Project: Maven
  • Language: Java
  • Spring Boot: 3.x (latest stable)
  • Group: com.example
  • Artifact: demo
  • Dependencies: Spring Web

Click Generate to download a ZIP file, then extract it.

Step 2: Explore the Project Structure

demo/
├── src/
│   ├── main/
│   │   ├── java/com/example/demo/
│   │   │   └── DemoApplication.java
│   │   └── resources/
│   │       └── application.properties
│   └── test/
│       └── java/com/example/demo/
│           └── DemoApplicationTests.java
├── pom.xml
└── mvnw

Step 3: Write a REST Controller

Create a new file HelloController.java in the same package:

package com.example.demo;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @GetMapping("/")
    public String home() {
        return "Welcome to my Spring Boot app!";
    }

    @GetMapping("/about")
    public String about() {
        return "This is a demo Spring Boot application.";
    }
}

Step 4: Run the Application

From the project root, run:

./mvnw spring-boot:run

You should see output ending with:

Started DemoApplication in 2.3 seconds

Open http://localhost:8080 in your browser and you'll see your response.

Step 5: Add application.properties

You can customize behavior in src/main/resources/application.properties:

# Change the default port
server.port=9090

# Set the application name
spring.application.name=my-demo-app

Step 6: Package as a JAR

To create a self-contained executable JAR:

./mvnw clean package
java -jar target/demo-0.0.1-SNAPSHOT.jar

This single JAR includes an embedded Tomcat server — no external application server needed.
